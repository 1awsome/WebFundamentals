# 影片 {: .page-title }

{% include "_shared/contributors/TODO.html" %}



使用者都喜歡影片，因為影片可以寓教於樂。在行動裝置中，影片可協助使用者輕鬆吸收資訊。但是，影片會佔用頻寬，而且在不同平台上的效果也可能有所差異。如果影片載入時間過久，或是按下播放鍵後卻沒有任何反應，都會讓使用者心生反感。請繼續讀下去，即可瞭解如何以最簡單的方式為網站新增影片，並確保使用者透過任何裝置都可獲得最佳體驗。
